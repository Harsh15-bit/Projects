CSS Tutorial

What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to style HTML elements on web pages. It controls the layout, colors, fonts, and overall visual appearance of a website.

Why Use CSS?

  • Separates content (HTML) from design (CSS).
  • Improves consistency across web pages.
  • Makes websites more responsive and visually appealing.
  • Reduces code repetition by using external stylesheets.

Types of CSS

There are three main types of CSS:

  • 1️⃣ Inline CSS
  • 2️⃣ Internal CSS
  • 3️⃣ External CSS

1️⃣ Inline CSS

Inline CSS is written directly inside the HTML tag using the style attribute. It is used for quick styling of individual elements.

<!DOCTYPE html>
<html>
<body>

<h1 style="color:blue; text-align:center;">
  Hello Inline CSS!
</h1>

</body>
</html>

Use Case: Best for quick testing or when applying a one-time style to an element.

2️⃣ Internal CSS

Internal CSS is placed inside the <style> tag within the <head> section of an HTML document.

<!DOCTYPE html>
<html>
<head>
<style>
  body {
    background-color: black;
    color: white;
  }
  h1 {
    color: cyan;
  }
</style>
</head>
<body>

<h1>Welcome to Internal CSS</h1>

</body>
</html>

Use Case: Ideal when you want to style a single page with unique formatting.

3️⃣ External CSS

External CSS is stored in a separate file with the .css extension. It is linked to the HTML file using the <link> tag.

/* style.css file */
body {
  background-color: #000;
  color: #fff;
}

h1 {
  color: lime;
  text-align: center;
}

HTML Link Example:

<!DOCTYPE html>
<html>
<head>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <h1>Welcome to External CSS</h1>
</body>
</html>

Use Case: Best for websites with multiple pages sharing the same styles.

CSS Syntax

The basic syntax of CSS consists of a selector and a declaration block.

selector {
  property: value;
}

Example:

h1 {
  color: red;
  font-size: 30px;
}

Example with All Types Combined

<!DOCTYPE html>
<html>
<head>
  <style>
    p { color: green; } /* Internal CSS */
  </style>
  <link rel="stylesheet" href="styles.css"> 
</head>
<body>

  <h1 style="color: blue;">Inline CSS Example</h1>
  <p>This is styled using internal CSS.</p>

</body>
</html>
