Python Programming Tutorial

Learn the powerful, easy, and versatile language that powers the modern world.

About Python

Python is a high-level, interpreted programming language created by Guido van Rossum in 1991. It emphasizes readability, simplicity, and productivity. It is widely used in web development, artificial intelligence, machine learning, data analysis, and automation.

History of Python

  • 1989: Python was first conceived by Guido van Rossum.
  • 1991: The first version (Python 0.9.0) was released.
  • 2000: Python 2.0 introduced new features like list comprehensions.
  • 2008: Python 3.0 was released with major improvements and better Unicode support.

Python File Types

  • .py → Regular Python source file
  • .pyc → Compiled Python file (bytecode)
  • .pyo → Optimized Python file (generated using -O flag)
  • .ipynb → Jupyter Notebook file (used for data science)

Basic Syntax

Python uses indentation (spaces) to define code blocks instead of curly braces.

# Example
if True:
  print("Hello, Python!")
else:
  print("Goodbye!")

Variables and Data Types

  • int → Integer numbers
  • float → Decimal numbers
  • str → Strings
  • bool → True/False values
  • list, tuple, set, dict → Collections
x = 10
y = 3.5
name = "Alice"
print(type(y))

Control Statements

Python provides conditional and looping statements to control the flow of programs.

# If-Else
age = 18
if age >= 18:
  print("You are an adult.")
else:
  print("You are a minor.")

# Loop
for i in range(5):
  print("Number:", i)

Functions

Functions help organize code into reusable blocks.

def greet(name):
  return f"Hello, {name}!"

print(greet("Harsh"))

Data Structures

  • List: Ordered and mutable → [1, 2, 3]
  • Tuple: Ordered and immutable → (1, 2, 3)
  • Set: Unordered, unique elements → {1, 2, 3}
  • Dictionary: Key-value pairs → {"name":"Alice", "age":25}
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
  print(fruit)

File Handling

Python makes it easy to read and write files.

# Writing to a file
f = open("data.txt", "w")
f.write("Hello, Python File!")
f.close()

# Reading from a file
f = open("data.txt", "r")
print(f.read())
f.close()

Sample Program

Example: Find the sum of all numbers in a list.

numbers = [10, 20, 30, 40, 50]
total = 0
for num in numbers:
  total += num
print("Sum =", total)
