üîπ What is JavaScript?

JavaScript (JS) is a lightweight, interpreted programming language used to make web pages interactive. It was created by Brendan Eich in 1995 and is now a core web technology alongside HTML and CSS.

üîπ Variables and Data Types

JavaScript supports var, let, and const for variable declarations.

// Example:
let name = "Harsh";
const age = 21;
var city = "Delhi";
console.log(name, age, city);

üîπ Operators and Conditional Statements

let num = 10;
if (num % 2 === 0) {
  console.log("Even Number");
} else {
  console.log("Odd Number");
}

üîπ Loops and Arrays

// Example of array and loop
let fruits = ["Apple", "Banana", "Cherry"];
for (let fruit of fruits) {
  console.log(fruit);
}

üîπ Functions

Functions allow code reuse. You can declare or use arrow functions.

// Normal Function
function greet(name) {
  return "Hello " + name;
}

// Arrow Function
const square = (n) => n * n;

console.log(greet("Harsh"));
console.log(square(5));

üîπ Objects and Classes (ES6)

// Object Example
let person = {
  name: "Riya",
  age: 22,
  greet() {
    console.log("Hello, " + this.name);
  }
};
person.greet();

// Class Example
class Car {
  constructor(brand, model) {
    this.brand = brand;
    this.model = model;
  }
  show() {
    console.log(`Car: ${this.brand} ${this.model}`);
  }
}

let c1 = new Car("BMW", "X5");
c1.show();

üîπ DOM Manipulation

DOM (Document Object Model) allows JavaScript to access and modify HTML content dynamically.

// Example: Changing HTML element text
document.getElementById("demo").innerHTML = "Text Changed!";

üîπ Events in JavaScript

Events are actions that happen in the browser, like clicks or key presses.

// Example
function showAlert() {
  alert("Button Clicked!");
}
