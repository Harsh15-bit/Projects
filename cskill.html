üîπ About C and C++

C is a structured programming language developed by Dennis Ritchie at Bell Labs in 1972. It focuses on procedures and is widely used for system software and embedded systems.

C++ was developed by Bjarne Stroustrup as an extension of C, adding Object-Oriented Programming (OOP) features.

üîπ Features and Properties of C

  • Procedural Programming Language
  • Portability and efficiency
  • Low-level memory manipulation
  • Rich library support
  • Fast compilation and execution

Example 1: Variables and Data Types

#include <stdio.h>
int main() {
    int age = 20;
    float height = 5.9;
    char grade = 'A';
    printf("Age: %d\\nHeight: %.1f\\nGrade: %c", age, height, grade);
    return 0;
}

Example 2: Arrays and Loops

#include <stdio.h>
int main() {
    int marks[5] = {85, 90, 78, 92, 88};
    int sum = 0;
    for (int i = 0; i < 5; i++) {
        sum += marks[i];
    }
    printf("Average Marks = %d", sum / 5);
    return 0;
}

Example 3: Functions and Conditional Statements

#include <stdio.h>
int max(int a, int b) {
    return (a > b) ? a : b;
}
int main() {
    int x = 15, y = 25;
    printf("Greater number is: %d", max(x, y));
    return 0;
}

üîπ Features of C++ (Object-Oriented Programming)

C++ extends C by introducing Object-Oriented Programming (OOP). The four pillars of OOP are:

  1. Encapsulation
  2. Inheritance
  3. Polymorphism
  4. Abstraction

1Ô∏è‚É£ Encapsulation

Encapsulation is the concept of wrapping data and methods into a single unit called a class. It helps protect data from unauthorized access.

#include <iostream>
using namespace std;

class Student {
  private:
    int rollNo;
    string name;
  public:
    void setData(int r, string n) {
        rollNo = r;
        name = n;
    }
    void display() {
        cout << "Roll No: " << rollNo << ", Name: " << name << endl;
    }
};

int main() {
    Student s;
    s.setData(101, "Harsh");
    s.display();
    return 0;
}

2Ô∏è‚É£ Inheritance

Inheritance allows one class to acquire properties and behaviors of another class.

#include <iostream>
using namespace std;

class Person {
  public:
    string name;
    int age;
    void displayPerson() {
        cout << "Name: " << name << ", Age: " << age << endl;
    }
};

class Student : public Person {
  public:
    int marks;
    void displayStudent() {
        displayPerson();
        cout << "Marks: " << marks << endl;
    }
};

int main() {
    Student s1;
    s1.name = "Aarav";
    s1.age = 20;
    s1.marks = 95;
    s1.displayStudent();
    return 0;
}

3Ô∏è‚É£ Polymorphism

Polymorphism allows the same function to behave differently based on the object that calls it.

#include <iostream>
using namespace std;

class Shape {
  public:
    virtual void draw() {
        cout << "Drawing a shape" << endl;
    }
};

class Circle : public Shape {
  public:
    void draw() override {
        cout << "Drawing a Circle" << endl;
    }
};

class Rectangle : public Shape {
  public:
    void draw() override {
        cout << "Drawing a Rectangle" << endl;
    }
};

int main() {
    Shape *s1;
    Circle c;
    Rectangle r;

    s1 = &c;
    s1->draw();
    s1 = &r;
    s1->draw();
    return 0;
}

4Ô∏è‚É£ Abstraction

Abstraction hides internal details and shows only essential features to the user.

#include <iostream>
using namespace std;

class Vehicle {
  public:
    virtual void start() = 0; // Pure virtual function
};

class Car : public Vehicle {
  public:
    void start() override {
        cout << "Car started with key ignition" << endl;
    }
};

int main() {
    Car c1;
    c1.start();
    return 0;
}
